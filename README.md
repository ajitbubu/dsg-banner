# dsg-banner

A lightweight cookie consent banner (CSS + JS) that can be dropped into any website.

## Files

dsg-cookie.css: styles for the cookie banner.
dsg-cookie.js: behavior/logic for the cookie banner.

## Usage

Include the files directly in your HTML using the raw GitHub URLs:

```html
<link rel="stylesheet" href="https://raw.githubusercontent.com/ajitbubu/dsg-banner/main/dsg-cookie.css">
<script src="https://raw.githubusercontent.com/ajitbubu/dsg-banner/main/dsg-cookie.js"></script>
```

Or, for correct MIME types and CDN caching, use the jsDelivr mirror:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ajitbubu/dsg-banner/dsg-cookie.css">
<script src="https://cdn.jsdelivr.net/gh/ajitbubu/dsg-banner/dsg-cookie.js"></script>
```
